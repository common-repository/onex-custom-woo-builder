== Description ==
Custom Woo Builder is a plugin that allows creating WooCommerce Single Product page templates and WooCommerce Product Archive templates with Elementor live page builder functionality using multiple dynamic content widgets.

With Custom Woo Builder it is easy to create the exact page structure one needs for the WooCommerce product or Archive page without ever touching a single line of code. The sections and columns can be built in seconds and filled up with content in a drag-n-drop way.

Use Custom Woo Builder to create unique WooCommerce Single Product page templates or Product Archive templates and use them for the products represented on Your site.

custom woo builder widgets also there like listed below: 

Single Add to Cart widget 
Single Attributes widget 
Single Excerpt widget 
Single Price widget
Single Rating widget 
Single Related Products widget
Single Reviews Form widget
Single Sale Badge widget
Single Sharing widget 
Single Title widget 
Single Upsells widget 

You can create custom Single Product Template, Product Archive Template, Shop Template & Catgeory Template from woocommerce > Custom Woo Templates.
Also Its provide some custom Glodal Widget called catgeories grid, products list, products grid & Taxnomany Tiles.

You have to allow just custom template for woocommerce page from woocommerce > settings > Custom Woo Builder

== Installation ==
1. Upload "custom-woo-bulider" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Place "do_action( 'plugin_name_hook' );" in your templates.
